# Kontaktbuch (Java)

Ein kleines Java-Programm zur Verwaltung einer Kontaktliste.  
- Kontakte anlegen  
- Kontakte anzeigen (sortiert)  
- Kontakte durchsuchen  
- Speicherung in `kontakte.txt`
